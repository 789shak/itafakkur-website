# If you discover a security vulnerability in iTafakkur, please:
#   1. Email support@itafakkur.com with the subject: "Security Report"
#   2. Include a clear description of the issue and, where safe,
#      reproduction steps.
#   3. Give us a reasonable window to remediate before public disclosure.
#
# We commit to acknowledging your report within 72 hours and
# resolving critical issues within 7 days where feasible.
#
# We are a solo-maintained project and do not currently offer a paid
# bug bounty, but we will publicly credit reporters (with consent) in
# our release notes.
#
